Let users add UWA widgets on their Wordpress dashboard.

== Description ==

The UWA Wordpress plugin let you add UWA widgets on your Wordpress dashboard. 

UWA is a simple widget format based on web standards such as HTML, CSS, XML and Javascript. It was originally developed by [Netvibes][], the personalized page pioneer.

You can find widgets on the [Netvibes Ecosystem][] directory.

[Netvibes]: http://www.netvibes.com/
[Netvibes Ecosystem]: http://eco.netvibes.com/

== Installation ==

This plugin follows the [standard WordPress installation method][]:

1. Make sure you have the [Zend Framework plugin][] installed and activated. It's available from Wordpress plugin directory.
1. Install the plugin
  - Automaticaly
  - or Manually: Upload the `uwa-widgets` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add new widgets through the 'UWA Widgets' section of the 'Options' menu

To find new widgets, you can browse [Netvibes Ecosystem][] directory. The URL you should use is the 'source' link on each widget page.

Some examples:

* http://znarf.h6e.net/netvibes/gapingvoid.html
* http://znarf.h6e.net/netvibes/dilbert.uwa.html
* http://www.netvibes.com/modules/digg/digg.html

[Zend Framework plugin]: http://wordpress.org/extend/plugins/zend-framework/
[standard WordPress installation method]: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
[Netvibes Ecosystem]: http://eco.netvibes.com/
